AVi - US Accidents Smart Visualizer

The AVi (US Accidents Smart Visualizer) webpage shows to the users data collection regardind USA car accidents between 2016-2019. The website delivers a variaty of charts with relevant stats of accidents.

Raw data can be collected from here.

The Interface

-is very simple and friendly, the website is responsive so it can be viewed from any screen, from mobile phones to laptops/monitors-



Server-side

-purpose is to get requests from users and retrieve the desired data in form of charts-
-the server will also convert the charts to different extensions when the user will request it-

Used Technologies

-FRONTEND: HTML, CSS, JAVASCRIPT, Google Charts-
- BACKEND: - PHP, AJAX, MySQL -
